# keytechKit

keytechKit is a iOS / OSX framework for accessing the keytech WebAPI. 

## Start building a project

Simply add the following to your Podfile using CocoaPods:
´´´ ruby
pod 'keytechkit', '~>0.1'
```

keytechKit works for iOS >=8 and OSX >=10.9
